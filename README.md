# AI/Spec-Driven Book on Robotics

Welcome to the AI/Spec-Driven Book on Robotics! This educational resource covers modern robotics frameworks including ROS 2, Gazebo/Unity, NVIDIA Isaac, and VLA systems.

## Modules

### Module 1: The Robotic Nervous System (ROS 2)
- **Status**: Complete
- **Topics**:
  - ROS 2 Core Concepts (Nodes, Topics, Services)
  - Python-to-ROS Control (rclpy)
  - URDF for Humanoid Robots
- **Prerequisites**: Basic Python knowledge, ROS 2 Humble environment
- **Learning Objectives**: Understand fundamental ROS 2 concepts and create basic robotic applications

## Development

This book is built with [Docusaurus](https://docusaurus.io/), a modern static website generator.

### Installation

```bash
npm install
```

### Local Development

```bash
npm start
```

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

### Build

```bash
npm run build
```

This command generates static content into the `build` directory and can be served using any static contents hosting service.
