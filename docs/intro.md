# Introduction

Welcome to the AI/Spec-Driven Book on Robotics! This educational resource covers modern robotics frameworks including ROS 2, Gazebo/Unity, NVIDIA Isaac, and VLA systems.

## Modules

- **Module 1**: The Robotic Nervous System (ROS 2) - Fundamentals of ROS 2 including Nodes, Topics, Services, Python control via rclpy, and URDF for humanoid robots
- **Module 2**: [Upcoming] - Simulation environments with Gazebo and Unity
- **Module 3**: [Upcoming] - NVIDIA Isaac robotics platform
- **Module 4**: [Upcoming] - Vision Language Action (VLA) models

This book follows a beginner-friendly approach with clear explanations, runnable examples, and hands-on exercises to help you master robotics development.
